# Installation

## Install

### Homebrew
```bash
brew install imdevan/bookmark/bookmark
```

### Arch (AUR)
```bash
yay -S bookmark
```

### Manual
```bash
just build
sudo just install
```

## Verify Installation

```bash
bookmark
```

## Customize

```bash
bookmark config init # init config

bookmark config
```
See [config docs for more details](/configuration)
